Instructions & Documentation

How to Use

  • Run the App: Launch TypingAssistant.exe. A console window will appear (you can minimize this).
  • Triggering: Type any word unaccented (e.g., coffee) and press the Insert key.
  • Dynamic Updates: The app checks for new versions every 5 minutes from our version.json server.
  • Instant Reload: Made changes to your dictionary? Press Ctrl + Alt + R to reload without restarting.

Editing Your Dictionary

Open dictionary.txt (or hotkeys.txt) and add your own shortcuts using the format: trigger=replacement.
